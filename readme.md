## Project Name

starter kit for vanilla with Webpack, tailwind, postcss, jquery

## Setup

Firstly clone the project $ git clone https://github.com/nuridemir/Webpack-Vanilla-Starter.git


npm install

npm update

npm run start

Build;

npm run dev

npm run build
